Política de Privacidad y Gestión de Datos

En esta aplicación, tus datos se gestionan de acuerdo con las mejores prácticas de privacidad.

  • Solo se almacenan los datos mínimos necesarios para la autenticación (correo y UID).
  • No se comparten datos personales con terceros.
  • El almacenamiento de sesión se gestiona mediante Firebase Authentication.
  • Puedes solicitar la eliminación de tus datos en cualquier momento.

Al continuar utilizando la aplicación, aceptas esta política de privacidad y la gestión de tus datos de acuerdo con estos términos.
